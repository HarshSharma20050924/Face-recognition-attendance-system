# OmniSight Demo - Quick User Guide

##  For Demo Users (Try It Now!)

### **First-Time Setup (2 Minutes)**

1. **Access the Demo:**
   - Go to: [https://face-recognition-attendance-system-true.vercel.app/](https://face-recognition-attendance-system-true.vercel.app/)
   - Click the **lock icon** on welcome screen

2. **Register Admin (First User Only):**
   - Stand 1-2 feet from webcam
   - Let camera capture your face clearly
   - Set a secure password
   - Click "Register Admin"

### **Admin Dashboard Tour**

| Section | What to Do |
|---------|------------|
| ** Dashboard** | View sample analytics & charts |
| **  Students** | Add sample students with photos |
| **  Faculty** | Register faculty members |
| ** Subjects** | Create sample subjects |
| ** Timetable** | Set up demo class schedule |

### **Quick Demo Flow**

#### **Step 1: Add Demo Data**
1. Go to **Students** → "Add Student"
2. Upload a clear face photo
3. Fill sample details (or use defaults)
4. Repeat for 3-4 students

#### **Step 2: Try Kiosk Mode**
1. Go to **Faculty** → "Add Faculty"
2. Register yourself as faculty
3. Go to **Kiosk Mode** (top-right corner)
4. **Face login** using your webcam
5. Select a subject from dropdown
6. Start attendance session

#### **Step 3: Mark Attendance**
1. Ask friends/family to stand before camera
2. Or use your own face for testing
3. System will auto-detect and mark attendance
4. Watch live updates on screen

#### **Step 4: Generate Reports**
1. End session (requires face verification)
2. Go to **Reports** section
3. Select session date
4. Download PDF/CSV report

### * Demo Tips**

**For Best Results:**
-  Good lighting (face well-lit)
-  Plain background
-  Face centered in frame
-  Remove glasses/hat if possible

**Quick Tests:**
- Try same person twice → System prevents duplicates
- Try different angles → System asks to face forward
- Try in low light → System gives feedback
- Try ending session → Requires face verification

### ** Reset Demo Data**

To start fresh:
1. Clear browser localStorage
2. Refresh page
3. Re-register as admin

### ** Mobile Testing**

Works on mobile too!
- Use front/back camera
- Portrait mode supported
- Touch-friendly interface
